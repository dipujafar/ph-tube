Discuss the scope of var, let, and const

Variables declared with var are in the function scope. Variables declared as let and const are in the block scope.

Tell us the use cases of null and undefined

Undefined means the variable has been declared, but its value has not been assigned. Null means an empty value or a blank value.

What do you mean by REST API?

What is REST API and how it works? A RESTful API is an architectural style for an application program interface (API) that uses HTTP requests to access and use data. That data can be used to GET, PUT, POST and DELETE data types, which refers to the reading, updating, creating and deleting of operations concerning resources.
